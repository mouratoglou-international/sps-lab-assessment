# TPS LAB — Sport Performance Science LAB
## Assessment Page

このリポジトリはTPS LABのオンラインパフォーマンス診断ページです。

### ファイル構成
```
tps-lab-assessment/
├─ index.html        ← ページ本体
├─ assets/
│   └─ logo.png     ← SPS_LAB_transparent.png をリネームして配置
└─ README.md
```

### セットアップ手順

1. `SPS_LAB_transparent.png` を `assets/logo.png` にリネームしてアップロード
2. `index.html` 内の `YOUR_FORM_ID` をFormspreeのIDに置き換える
   - 例: `https://formspree.io/f/xyzabcde`
3. GitHub Pages を有効化（Settings → Pages → main branch）
